用户 API

登录:POST /1/users/login
参数:email,password
返回值:{code:code,message:'message'} code=1表示出错，code=0 表示成功 cookie里有登录认证信息（以后同此）

登录:GET /1/users/logout
参数:
返回值:{code:code,message:'message'}

登录:POST /1/users/register
参数:email,password,username,gender
返回值:{code:code,message:'message'}

登录:POST /1/users/modifypwd
参数:oldPwd,password
返回值:{code:code,message:'message'}

登录:GET /1/users/avatar/aid(头像图片id)
参数:
返回值:图片二进制

登录:POST /1/users/avatar
参数:avatar(文件)
返回值:{code:code,message:'message'}

说说 API

发布:POST /1/voices/new
参数::messageType('text' or 'mp3') 如果是text,则需要message参数，如果是mp3则需要文件voice，还有播放长度秒 还有经度longitude，纬度latitude
返回值:{code:code,message:'message'}

删除:GET /1/voices/delete/:vid
参数::vid
返回值:{code:code,message:'message'}

用户说说列表:GET /1/voices/list/:uid/:page
参数::vid,:page
返回值:{code:code,message:'message'}

用户说说数目:GET /1/voices/count/:uid
参数::uid
返回值:{code:code,message:'message'}

说说回复:POST /1/voices/doreply/:vid
参数::vid，:messageType('text' or 'mp3') 如果是text,则需要message参数，如果是mp3则需要文件voice，还有播放长度秒
返回值:{code:code,message:'message'}

回复列表:GET /1/voices/reply/:vid/:page
参数::vid,:page
返回值:{code:code,message:'message'}

删除回复:GET /1/voices/delreply/:rid
参数::rid
返回值:{code:code,message:'message'}

回复数目:GET /1/voices/replycount/:vid
参数::vid
返回值:{code:code,message:'message'}

语音实体信息:GET /1/voices/show/:vid
参数::vid
返回值:{code:code,message:'message'}

说说实体:GET /1/voices/of/:vid
参数::vid
返回值:{code:code,message:'message'}
